# vue3-rick-and-morty (Vue 3 + Vite)

Demo: https://alefiengo-vue3-rick-and-morty.netlify.app

## Requisitos
- Node.js 22 (LTS)

## Ejecutar en local
```
npm install
npm run dev
```

## Build
```
npm run build
```

## Preview del build
```
npm run preview
```

## Deploy (Netlify)
- Configurado en `netlify.toml` (build: `npm run build`, publish: `dist`).
