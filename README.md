# vue3-rick-and-morty

## Deploy on Netlify
Visit [https://alefiengo-vue3-rick-and-morty.netlify.app](https://alefiengo-vue3-rick-and-morty.netlify.app)

## Project setup
```
npm install
```

### Notes (Node 17+ / OpenSSL 3)
This project uses Vue CLI 4 / Webpack 4; the `serve`/`build` scripts include `NODE_OPTIONS=--openssl-legacy-provider` so it works on newer Node versions.

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
